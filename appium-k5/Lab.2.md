```
input: 9

*********

*******

****
```

## Finding indices 
* Given an array
* Allow user to input a target number
* Output: print out all pair of element indexes those are sum up to the target number
* Example:
```
int [] myIntArr = {1,2,3,4,5,6};
int inputNum = 7;
output:
(0, 5)
(1, 4)
(2, 3)
```
