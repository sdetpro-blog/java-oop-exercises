## Lab 2.1:
Get input from user about height(m) and weight(kg) then calculate BMI
- Underweight = <18.5
- Normal weight = 18.5 – 24.9
- Overweight = 25–29.9
- Obesity = BMI of 30 or greater

```
BMI = weight / (height x 2)
```
